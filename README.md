# ⚽ Fußball Legenden - Professionelle Sportwebseite

Eine moderne, interaktive und visuell beeindruckende Webseite, die die Geschichte von fünf der größten Fußballvereine Europas erzählt.

## 🌟 Features

### Vereine
- **FC Bayern München** - Deutscher Gigant mit 6 Champions League Titeln
- **Real Madrid** - Spanischer König mit 14 Champions League Titeln (Rekord!)
- **FC Barcelona** - Der Tiki-Taka Meister aus Spanien
- **Manchester United** - Englische Legende seit 1878
- **Juventus Turin** - Italienischer Klassiker seit 1897

### Design & UX
✅ **Modernes responsives Design** - Perfekt auf Desktop, Tablet und Mobile  
✅ **Dunkles sportliches Farbschema** - Mit dynamischen Akzentfarben  
✅ **Hero-Section** - Beeindruckender Einstieg mit Animationen  
✅ **Smooth Scrolling** - Elegante Seitennavigation  
✅ **Große Bildergalerien** - Mit Hover-Effekten  

### Interaktive Elemente
✅ **Quiz-Sektion** - Teste Dein Wissen über Fußballgeschichte  
✅ **Vergleichstabelle** - Mit sortierbar Spalten  
✅ **Hover-Effekte** - 3D-Kartenbewegungen  
✅ **Scroll-Animationen** - Elemente laden elegant beim Scrollen  
✅ **Mobile Menu** - Hamburger-Navigation für Mobilgeräte  

### Für jeden Verein
📅 **Gründungsjahr**  
🏆 **Erfolge & Titel** (Champions League, Liga-Meisterschaften)  
🏟️ **Stadioninformationen**  
👥 **Legendäre Spieler** (Mit 4 Spielern pro Verein)  
📈 **Zeitleiste der Vereinsgeschichte**  
🖼️ **Bildergalerie** (6 Bilder pro Verein)

## 🚀 Features in Detail

### 1. Navigation
- Sticky Navbar mit Gold-Akzent
- Aktiver Link-Indicator
- Responsive Hamburger-Menü
- Schnelle Sprunglinks zu jedem Verein

### 2. Hero-Section
- Vollbild Hero-Image
- Animierter Fußball
- Farbgradienten
- Call-to-Action Button mit Hover-Effekt

### 3. Vereinssektionen
Für jeden Verein:
- **Header** mit Logo und Vereinsinfo
- **Info-Karten** mit wichtigen Daten
- **Interaktive Timeline** der Vereinsgeschichte
- **Spielerkarten** mit 3D-Hover-Effekt
- **Bildergalerie** mit Overlay-Text

### 4. Vergleichstabelle
- Sortierbare Spalten (klickbar)
- Alle 5 Vereine auf einen Blick
- Farbkodierung der Reihen
- Responsive Ausführung

### 5. Quiz
- 3 Fragen zu Fußballgeschichte
- Sofortiges Feedback
- Grüne Hervorhebung bei Richtig
- Rote Hervorhebung bei Falsch
- Toast-Benachrichtigungen

### 6. Animationen
- **Fade-In Animationen** beim Scrollen
- **Parallax-Effekt** für Tiefenwirkung
- **Pulse-Animation** auf Cards
- **Floating Animation** im Hero-Bereich
- **3D-Rotation** bei Card-Hover
- **Shimmer-Effekt** auf Elementen

## 📱 Responsive Design

Die Website ist vollständig responsive:

- **Desktop** (> 1200px): Volle Funktionalität mit allen Features
- **Tablet** (768px - 1200px): Angepasstes Layout, lesbar
- **Mobile** (< 768px): Optimiert für kleine Bildschirme
  - Hamburger-Menü statt Menüleiste
  - Einspaltige Layouts
  - Optimierte Schriftgrößen
  - Touch-freundliche Buttons

## 🎯 Technologien

- **HTML5**: Semantische Struktur
- **CSS3**: Moderne Styling, Animationen, Flexbox, Grid
- **JavaScript (Vanilla)**: Keine Abhängigkeiten
- **Google Fonts**: Poppins, Playfair Display, Inter
- **Font Awesome**: Icon-Library
- **Intersection Observer API**: Für Scroll-Animationen

## 📝 Verwendung

1. **Öffne die Datei**:
   - Klicke einfach auf `index.html` im Explorer
   - Oder öffne über "Open with Live Server" (VSCode Extension)

2. **Navigiere die Website**:
   - Nutze das Menü oben oder die Schnelllinks
   - Klicke auf "Entdecke die Vereine" im Hero
   - Scroll für parallax Effekte

3. **Interaktivität erkunden**:
   - Hover über Cards für 3D-Effekte
   - Probiere das Quiz
   - Sortiere die Vergleichstabelle
   - Teste Mobile-Ansicht (F12 → Responsive Design)
